# Secret

Secret 프로그램은 저평가된 주식을 찾을 수 있도록 정보를 제공합니다. 현재는 대한민국 주식 및 ETF 정보만 제공합니다. 추후 유용한 정보들을 추가해 나갈 예정입니다.

! 경고 : 이 프로그램에서 제공하는 투자정보는 부정확할 수 있으므로 투자판단에 참고만 하시기 바랍니다. 투자에 대한 책임은 투자자 '본인'에게 있습니다.

## Support OS

- Windows 7 이상
